# FIFA2026 專案設定

純 HTML / CSS / JavaScript 前端專案，無框架、無建置工具。

## 技術限制

- 不使用任何前端框架（Vue、React 等）
- 不引入 npm 套件，除非明確要求
- 相容性目標：現代瀏覽器（Chrome、Firefox、Edge 最新版）

## 檔案慣例

- HTML：`index.html` 為進入點
- 樣式放 `css/`，腳本放 `js/`，圖片放 `assets/`
- JS 使用 ES6+ 原生語法（`const`、`fetch`、`async/await`）

## 開發測試

用瀏覽器直接開啟 HTML 檔，或 `npx serve .` 起本地伺服器。
