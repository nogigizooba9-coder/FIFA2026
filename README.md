# FIFA 2026 活動頁

FIFA 2026 世界盃主題活動頁面，供玩家使用，嵌入平台展示。  
純 HTML / CSS / JavaScript 開發，無框架依賴。

## 功能區塊

| # | 名稱 | 說明 |
|---|------|------|
| 1 | 賽事排程 | 顯示 FIFA 2026 比賽時程 |
| 2 | 競猜預測贈彩金 | 玩家預測比賽結果，命中可獲彩金獎勵 |
| 3 | 下注世足簽到彩金 | 賽事期間每日簽到下注，領取彩金 |
| 4 | 全站續儲贈彩金 | 活動期間連續儲值享彩金加成 |
| 5 | 投注排名直播抽獎 | 依投注金額排名，直播抽獎活動 |

## Markdown 文字標記規則

活動規則文字支援以下自訂標記，可在各頁面的 `defaultMarkdown` 字串中使用。

| 標記語法 | 效果 |
|---|---|
| `**文字**` | 粗體 |
| `[red]文字[/red]` | 紅色字體 |
| `[red-bold]文字[/red-bold]` | 紅色字體 + 粗體 |

**範例**

```
1.一般條文，無特別標記。
2.**這段文字會顯示為粗體。**
3.請注意 [red]這段重要警示會顯示為紅色[/red]，請仔細閱讀。
4.[red-bold]這段同時為紅色加粗體[/red-bold]，用於最高警示。
```

> 建議只用在活動規則或重點警示，避免整頁全部變紅造成閱讀疲勞。
